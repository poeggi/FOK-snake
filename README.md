# Snake - FOK Edition

Classic Snake in the browser. 10 levels, arcade music, high scores.

**Play online:** https://poeggi.github.io/FOK/

## Controls

| Key | Action |
|-----|--------|
| Arrow keys | Move snake / navigate menus |
| Enter | Select / confirm |
| Backspace | Delete (name entry) |
| A-Z keys | Type name directly (name entry) |
| Esc | Back (settings) |

## Features

- 10 levels - speed and barricades increase each level
- Snake starts longer in advanced levels
- Screen wraps on all edges
- 3 lives - hit a barricade or your own tail, lose one
- Collect 10 gems per level to advance
- Arcade chiptune music (Web Audio API, no downloads)
- High score table saved locally
- Easy / Normal / Hard difficulty
- FPS counter
